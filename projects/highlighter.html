Highlighter

A modern web-based text highlighting and annotation tool

Overview

Highlighter is a collaborative web platform that lets users mark up, annotate, and share important sections of any text. Built for researchers, writers, and teams who need a frictionless way to capture and communicate ideas within documents.

Users can highlight passages in multiple colours, attach inline comments, and share annotated documents via a unique link — no account required for viewers. Real-time collaboration means multiple users can annotate the same document simultaneously, with changes appearing instantly.

The editor supports plain text and rich content, with an export option that preserves all highlights and comments as a formatted PDF or HTML file.
