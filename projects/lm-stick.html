LM Stick

A local LLM management and deployment platform

Overview

LM Stick is a desktop platform for running, managing, and switching between large language models locally. It addresses the friction of working with multiple open-source models — different quantisations, context lengths, and prompt formats — by providing a unified interface for the entire workflow.

Models are organised into a library with metadata, benchmarks, and usage history. Switching between models is a single click, with automatic context preservation where architectures allow. A built-in prompt playground lets you test and compare model outputs side by side.

LM Stick exposes a local OpenAI-compatible API endpoint, meaning any tool or script that works with the OpenAI SDK works with LM Stick out of the box — no code changes required.
